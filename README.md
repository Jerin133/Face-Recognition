# Face-Recognition
A simple Face recognition system using python tkinter. It is working but just like a prototype.
This Face recognation system project was my 3rd sem project so it will not very clear.But it will recognize the single face if the picture it capture was clear.Here it start with login page,it has a default username and password which was set in the program.Then the main page of the face recognition system will be opened.this page has 3 icons new register,face detector and the list.In new register there will be the options saving new registeration,update photo and take photo.In the list I used the normal csv file set up without using sql.And it was little messy with the face detector.
